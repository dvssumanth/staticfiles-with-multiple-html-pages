Example of Change the background color of page

to change the background color of page, apply background-color property in style attribute.
Ex: <div style="background-color: red;"></div>

Example of Scroll Down

Please scroll down the page. Now, you can able to see 'Go to top' link. Click on that you will navigate to top of the page.
Ex: <a href="#top">Go to top</a>
In above anchor ( a ) tag contains href attribute. it navigates you to specified id given by you. the id which is placed in top of the page like as id="top" at html tag.
